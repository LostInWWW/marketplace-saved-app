# Marketplace Saved Organizer v6

This version adds a compact iPhone import mode to reduce Safari storage problems.

## New in v6

- Compact iPhone import is on by default
- Keeps stronger records first:
  - titled items with Marketplace URLs
  - collection items
  - a limited slice of titled listing-history items
- Clear notes and flags button
- Summary now shows whether compact import is on
- Fixed CSV export

## Recommended settings for iPhone

- Status: All statuses
- Titled items only: on
- Marketplace URL only: on
- Compact iPhone import: on

## Upload to GitHub Pages

Replace these files in your repo root:

- index.html
- app.js
- style.css
- README.md

Then wait for GitHub Pages to refresh and reload the site.
